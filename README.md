# Predicting Liver Cirrhosis Survival Analysis

## Overview
This project uses machine learning techniques to predict liver cirrhosis survival based on patient clinical data.

## Problem Statement
Liver cirrhosis is a serious health condition that requires early diagnosis and treatment. This project aims to analyze patient records and predict survival outcomes.

## Dataset
The dataset includes patient medical records such as bilirubin, albumin, platelets, and treatment variables.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Survival Analysis

## Project Workflow
1. Data Cleaning
2. Exploratory Data Analysis
3. Feature Engineering
4. Survival Curve Analysis
5. Model Building
6. Performance Evaluation

## Key Insights
- Treatment interactions with biomarkers significantly affect survival probability.
- Bilirubin and Albumin showed strong predictive power.

## Results
The project successfully identified important factors influencing liver cirrhosis survival.
